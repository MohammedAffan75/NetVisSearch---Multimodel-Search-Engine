From upload to insight in seconds

Your data stays local. The intelligence is everywhere.

01

Upload Anything

Drop in images, scanned PDFs, medical reports, or photos. EasyOCR reads the text even from messy scans, while Jina-CLIP-v2 understands visual context.

02

AI Understands

Every file is compressed into a 512-dimensional numeric fingerprint. Named entities (People, Places, Organizations) are extracted automatically using spaCy NLP.

03

Explore the Network

Search in plain English to see ranked results. A Three.js 3D force graph visualizes how your files relate through shared entities—uncovering connections you never knew existed.

Natural Language Search

Ask "show me photos from my trip" or "medical report from March". Matches meaning, not just keywords.

Multimodal Understanding

Encodes both visual images and OCR text into the same space. Your images and documents coexist intelligently.

3D Entity Network

Shared names, dates, and places link documents automatically. Navigate the graph to discover deeper insights.

DuckDB Vector Search

HNSW index on FLOAT[512] columns. Full SQL joins + ANN search in one query. Faster than any standard vector DB.

Zero Cloud Dependency

Every model runs locally. No API keys, no subscriptions, no data leaks. Your files never leave your drive.

Built cloud-ready. Deploy to AWS ECS or run on your home server with a single Docker command.
